# Порція — Android застосунок

Трекер калорій по фото: AI розпізнає їжу (Ollama на ноуті), USDA дає точні
цифри й вітаміни, є пошук страв, штрих-коди (Open Food Facts) і «Мої страви».

## Як зібрати APK через GitHub (у хмарі)

1. Створи новий репозиторій на github.com (кнопка **New**). Назва: `portsiya`.
   Постав **Public**, нічого більше не додавай, натисни **Create repository**.

2. Завантаж усі файли з цієї папки в репозиторій:
   на сторінці репозиторію → **Add file → Upload files** →
   перетягни ВЕСЬ вміст папки (разом з папками `www` і `.github`) →
   **Commit changes**.

3. Збірка почнеться сама. Відкрий вкладку **Actions** зверху —
   побачиш процес «Build APK». Зачекай ~5–8 хвилин (перший раз довше).

4. Коли з'явиться зелена галочка — натисни на завершений запуск,
   унизу в розділі **Artifacts** буде файл **portsiya-apk**. Завантаж його.

5. Розпакуй — усередині `app-debug.apk`. Це і є застосунок.

## Встановлення на телефон

1. Перекинь `app-debug.apk` на телефон (Telegram, пошта тощо).
2. Відкрий його. Android спитає дозвіл встановлювати з невідомих джерел — дозволь.
3. Встанови. З'явиться іконка «Порція».

## Перший запуск

Застосунок так само працює з Ollama на ноуті:
1. Ноут увімкнений, Ollama запущена (зі змінними OLLAMA_HOST=0.0.0.0 і OLLAMA_ORIGINS=*).
2. Телефон і ноут на одному Wi-Fi.
3. У застосунку ⚙️ → Адреса Ollama: http://IP_НОУТА:11434, модель: llava:7b.
4. Ключ USDA (за бажанням, для лімітів) — з api.data.gov/signup.

Сервер (start-server.bat) більше НЕ потрібен — застосунок сам містить усі файли.
